## get_balance

Get token balances for the authenticated wallet.

| Param   | Type   | Required |
| ------- | ------ | -------- |
| `token` | string | No       |

Omit `token` to fetch all non-zero balances. Specify a symbol (e.g., `STRK`, `ETH`) to check a single token.

---

## send_tokens

Transfer tokens to a recipient address.

| Param       | Type    | Required |
| ----------- | ------- | -------- |
| `amount`    | string  | Yes      |
| `token`     | string  | Yes      |
| `recipient` | string  | Yes      |
| `simulate`  | boolean | No       |

Set `simulate: true` to estimate fees without sending a transaction.

---

## deploy_account

Deploy the smart contract wallet on-chain.

**Parameters:** None

Required once before sending any transaction. Safe to call multiple times — returns the current status if already deployed.

---

## get_portfolio

Get a full DeFi overview: all token balances with USD values, staking positions, and lending positions.

**Parameters:** None

---

## get_tx_status

Check the status of a transaction by hash.

| Param  | Type   | Required |
| ------ | ------ | -------- |
| `hash` | string | Yes      |

Returns the current status, actual fee, block number, and explorer link.
