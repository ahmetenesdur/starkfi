## get_swap_quote

Get a price quote for a swap without executing. Always call this before `swap_tokens` so the user can review the expected output.

| Param        | Type   | Required |
| ------------ | ------ | -------- |
| `amount`     | string | Yes      |
| `from_token` | string | Yes      |
| `to_token`   | string | Yes      |

---

## swap_tokens

Execute a token swap on Starknet using Fibrous DEX aggregation. Call `get_swap_quote` first to show the user a preview.

| Param        | Type    | Required        |
| ------------ | ------- | --------------- |
| `amount`     | string  | Yes             |
| `from_token` | string  | Yes             |
| `to_token`   | string  | Yes             |
| `slippage`   | number  | No (default: 1) |
| `simulate`   | boolean | No              |

---

## get_multi_swap_quote

Get quotes for 2–3 swap pairs at once using Fibrous batch routing.

| Param   | Type  | Required |
| ------- | ----- | -------- |
| `swaps` | array | Yes      |

Each swap item: `{ amount, from_token, to_token }`

---

## multi_swap

Execute 2–3 swaps in a single multicall transaction.

| Param      | Type    | Required |
| ---------- | ------- | -------- |
| `swaps`    | array   | Yes      |
| `slippage` | number  | No       |
| `simulate` | boolean | No       |

Each swap item: `{ amount, from_token, to_token }`

> **Note:** Minimum 2, maximum 3 swap pairs per transaction.

---

## batch_execute

Execute a batch of mixed operations (swap + stake + supply + send) atomically.

| Param        | Type    | Required |
| ------------ | ------- | -------- |
| `operations` | array   | Yes      |
| `simulate`   | boolean | No       |

Each operation has a `type` and `params`:

| Type     | Params                                   |
| -------- | ---------------------------------------- |
| `swap`   | `amount`, `from_token`, `to_token`       |
| `stake`  | `amount`, `token`, `pool` or `validator` |
| `supply` | `amount`, `token`, `pool`                |
| `send`   | `amount`, `token`, `to`                  |

> **Note:** Minimum 2 operations required per batch.
