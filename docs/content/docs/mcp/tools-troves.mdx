## Overview

Troves MCP tools let AI agents interact with DeFi yield vault strategies on Starknet. Strategies range from single-asset vaults to LP strategies and leveraged positions тАФ all audited and managed by the Troves protocol.

> **Agent tip:** Always call `list_troves_strategies` to validate a strategy exists and check its risk profile before depositing.

## Tools

### `list_troves_strategies`

| Property | Value |
| --- | --- |
| **Type** | read |
| **Description** | List all active Troves vault strategies with APY, TVL, and risk tier |

**Input Schema:**

No required parameters.

**Response:**

```json
{
  "strategies": [
    {
      "id": "evergreen_strk",
      "name": "STRK Evergreen",
      "apy": "5.23%",
      "tvlUsd": "$2.02K",
      "depositTokens": ["STRK"],
      "riskFactor": 1.1,
      "isAudited": true,
      "status": "Active"
    }
  ]
}
```

## Agent Best Practices

1. **List before depositing** тАФ Call `list_troves_strategies` to verify the strategy exists and check current APY/risk
2. **Check existing position** тАФ Call `get_troves_position` before depositing to see current exposure
3. **Simulate first** тАФ Set `simulate: true` on deposit/withdraw to preview fees
4. **Confirm with user** тАФ Always get explicit confirmation before executing write operations
