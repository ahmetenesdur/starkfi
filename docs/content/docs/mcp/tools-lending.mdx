title: Lending Tools
description: 7 tools for Vesu V2 lending operations

## list_lending_pools

List all available Vesu V2 lending pools with APY data.

**Parameters:** None

---

## get_lending_position

Get your position in a lending pool.

| Param             | Type   | Required |
| ----------------- | ------ | -------- |
| `pool`            | string | Yes      |
| `collateralToken` | string | Yes      |
| `debtToken`       | string | Yes      |

Returns collateral value, debt value, health factor, and risk level.

---

## supply_assets
