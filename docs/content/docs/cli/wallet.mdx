## address

Shows your authenticated wallet address.

```bash
starkfi address
```

Shows your Starknet contract address and current network.

## balance

Shows all token balances in your wallet.

```bash
starkfi balance
```

Displays each token with its current balance. Use `--json` for machine-readable output.

## send

Sends tokens to another address.

```bash
starkfi send <amount> <token> <recipient>
```

**Arguments:**

| Argument    | Description                                |
| ----------- | ------------------------------------------ |
| `amount`    | Amount to send (e.g., `100`, `0.5`)        |
| `token`     | Token symbol (e.g., `ETH`, `STRK`, `USDC`) |
| `recipient` | Destination Starknet address               |

**Example:**

```bash
starkfi send 100 STRK 0x0123...abcd
```

## deploy

Deploys your smart contract wallet to the network.

```bash
starkfi deploy
```

Deploys the ArgentX v0.5.0 account contract if it hasn't been deployed yet. This is required before executing any transaction.
