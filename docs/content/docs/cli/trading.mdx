## trade

Swap one token for another using the best available route.

```bash
starkfi trade <amount> <fromToken> <toToken> [flags]
```

**Arguments:**

| Argument    | Description              |
| ----------- | ------------------------ |
| `amount`    | Amount to swap           |
| `fromToken` | Source token symbol      |
| `toToken`   | Destination token symbol |

**Flags:**

| Flag         | Default | Description                 |
| ------------ | ------- | --------------------------- |
| `--slippage` | `1`     | Maximum slippage percentage |
| `--simulate` | `false` | Dry-run mode                |

**Example:**

```bash
starkfi trade 0.5 ETH USDC --slippage 0.5
```

## multi-swap

Execute up to 3 swap pairs in a single multicall transaction.

```bash
starkfi multi-swap --pairs "<amount1> <from1> <to1>, <amount2> <from2> <to2>"
```

**Example:**

```bash
starkfi multi-swap --pairs "0.1 ETH USDC, 100 STRK USDC, 50 DAI USDC"
```

> **Note:** Maximum 3 swap pairs per transaction.

## status

Check the status of a submitted transaction.

```bash
starkfi tx-status <txHash>
```
