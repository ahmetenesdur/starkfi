---
title: Lending Commands
description: Supply, borrow, repay, and manage Vesu V2 lending positions
---

## lend-pools

Lists all available Vesu V2 lending pools.

```bash
starkfi lend-pools [name]
```

Shows pool name, protocol version, available assets, and pair count. Optionally filter by name (partial match — shows detailed asset info).

**Examples:**

```bash
starkfi lend-pools
starkfi lend-pools prime
starkfi lend-pools --json
```

## lend-supply

Supplies assets into a lending pool.

```bash
starkfi lend-supply <amount> -p <pool> -t <token>
```

**Flags:**

| Flag          | Description                                   |
| ------------- | --------------------------------------------- |
| `-p, --pool`  | Pool name (e.g., `Prime`) or contract address |
| `-t, --token` | Token symbol to supply (e.g., `USDC`, `ETH`)  |

**Example:**

```bash
starkfi lend-supply 100 -p Prime -t USDC
```

> **Note:** Minimum supply is approximately $10 equivalent (dust limit protection).

## lend-withdraw

Withdraws supplied assets from a pool.

```bash
starkfi lend-withdraw <amount> -p <pool> -t <token>
```

**Example:**

```bash
starkfi lend-withdraw 50 -p Prime -t USDC
```

## lend-borrow

Borrows assets against your collateral. Atomically deposits collateral and borrows the debt asset.

```bash
starkfi lend-borrow -p <pool> --collateral-amount <n> --collateral-token <symbol> --borrow-amount <n> --borrow-token <symbol>
```

**Flags:**

| Flag                  | Description                                        |
| --------------------- | -------------------------------------------------- |
| `-p, --pool`          | Pool name or contract address                      |
| `--collateral-amount` | Amount of collateral to deposit                    |
| `--collateral-token`  | Collateral token symbol (e.g., `ETH`, `STRK`)      |
| `--borrow-amount`     | Amount to borrow                                   |
| `--borrow-token`      | Token to borrow (e.g., `USDC`, `USDT`)             |
| `--use-supplied`      | Use previously supplied yield tokens as collateral |

**Example:**

```bash
starkfi lend-borrow -p Prime --collateral-amount 0.5 --collateral-token ETH --borrow-amount 500 --borrow-token USDC
```

## lend-repay

Repays borrowed debt.

```bash
starkfi lend-repay <amount> -p <pool> -t <token> --collateral-token <symbol>
```

**Flags:**

| Flag                 | Description                                    |
| -------------------- | ---------------------------------------------- |
| `-p, --pool`         | Pool name or contract address                  |
| `-t, --token`        | Token to repay (e.g., `USDC`)                  |
| `--collateral-token` | Collateral token of the position (e.g., `ETH`) |

**Example:**

```bash
starkfi lend-repay 500 -p Prime -t USDC --collateral-token ETH
```

## lend-close

Closes an entire lending position (repay all debt + withdraw all collateral).

```bash
starkfi lend-close -p <pool> --collateral-token <symbol> --borrow-token <symbol>
```

**Flags:**

| Flag                 | Description                           |
| -------------------- | ------------------------------------- |
| `-p, --pool`         | Pool name or contract address         |
| `--collateral-token` | Collateral token symbol (e.g., `ETH`) |
| `--borrow-token`     | Borrowed token symbol (e.g., `USDC`)  |

**Example:**

```bash
starkfi lend-close -p Prime --collateral-token ETH --borrow-token USDC
```

## lend-status

Checks your lending position in a pool.

```bash
starkfi lend-status -p <pool> --collateral-token <symbol> [--borrow-token <symbol>]
```

**Flags:**

| Flag                 | Description                                             |
| -------------------- | ------------------------------------------------------- |
| `-p, --pool`         | Pool name or contract address                           |
| `--collateral-token` | Token supplied to the pool (e.g., `ETH`)                |
| `--borrow-token`     | Borrow token (optional — required to see debt position) |

Shows supplied yield balance, collateral value, debt value, health factor, and risk level:

| Risk Level  | Health Factor |
| ----------- | ------------- |
| **SAFE**    | Above 1.5     |
| **WARNING** | 1.1 – 1.5     |
| **DANGER**  | Below 1.1     |
