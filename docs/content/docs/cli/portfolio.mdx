## Usage

```bash
starkfi portfolio
```

## What It Shows

The portfolio command aggregates your complete DeFi state into a single view.

### Token Balances

All tokens with non-zero balances, each with USD value calculated via Fibrous pricing.

### Staking Positions

For each validator pool:

- Staked amount and USD value
- Pending rewards
- Unpooling status (if an exit intent exists)

### Lending Positions

For each Vesu V2 pool:

- Collateral value
- Debt value
- Health factor and risk level

### Total Portfolio Value

Sum of all token balances, staked amounts, and collateral in USD.

## JSON Output

For programmatic use or piping to other tools:

```bash
starkfi portfolio --json
```
