## Requirements

- **Node.js** 18 or later
- **npm**, **pnpm**, or **yarn**

## Quick Install

The fastest way to get started — install globally and run from anywhere:

```bash
npm i -g starkfi@latest
```

Verify it works:

```bash
starkfi --version
```

## Run Without Installing

Don't want to install? Use `npx` to run any command directly:

```bash
npx starkfi@latest trade 0.1 ETH USDC
```

This downloads the latest version on-the-fly. Great for quick one-off commands or trying StarkFi for the first time.

## Other Package Managers

```bash tab="npm"
npm i -g starkfi@latest
```

```bash tab="pnpm"
pnpm add -g starkfi@latest
```

```bash tab="yarn"
yarn global add starkfi@latest
```

## What Gets Installed

The `starkfi` package ships everything in a single install:

| Component        | What It Includes                                                  |
| ---------------- | ----------------------------------------------------------------- |
| **CLI**          | 30+ commands for trading, staking, lending, and wallet management |
| **MCP Server**   | 27 typed tools — start with `starkfi mcp-start`                   |
| **Agent Skills** | 10 pre-packaged workflows in the `skills/` folder                 |

## Verify Installation

```bash
# Show all available commands
starkfi --help

# Check current configuration
starkfi config list
```

## Next Step

After installation, **[authenticate your wallet](/docs/authentication)** to start using StarkFi.
