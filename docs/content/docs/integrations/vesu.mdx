## Overview

[Vesu](https://vesu.xyz) provides V2 lending pools on Starknet. StarkFi integrates directly with Vesu's on-chain contracts for full lending lifecycle management.

## Supported Pools

| Pool                 | Description                 |
| -------------------- | --------------------------- |
| Prime                | Primary multi-asset pool    |
| Re7 USDC Core        | Core USDC lending pool      |
| Re7 USDC Prime       | Premium USDC lending        |
| Re7 USDC Frontier    | Frontier USDC lending       |
| Re7 xBTC             | Bitcoin-denominated lending |
| Re7 USDC Stable Core | Stablecoin-focused lending  |

## Operations

| Operation | Contract Method          | Description            |
| --------- | ------------------------ | ---------------------- |
| Supply    | `vToken.deposit()`       | ERC-4626 vault deposit |
| Withdraw  | `vToken.withdraw()`      | Direct withdrawal      |
| Borrow    | `pool.modify_position()` | Add collateral + debt  |
| Repay     | `pool.modify_position()` | Reduce debt            |
| Close     | `pool.modify_position()` | Remove all             |
| Query     | `pool.position()`        | Read position state    |

## Health Factor

StarkFi calculates health factor as:

```
healthFactor = (collateralUSD × maxLTV) / debtUSD
```

Risk levels:

- **SAFE**: health factor > 1.5
- **WARNING**: 1.1 – 1.5
- **DANGER**: < 1.1

## Technical Details

- **API Base**: `https://api.vesu.xyz` (cached for 5 minutes)
- **Dust Limit**: Minimum ~$10 equivalent for supply/borrow
- **U256 Encoding**: Amounts are split into low/high 128-bit parts for Cairo calldata
